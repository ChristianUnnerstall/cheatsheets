#### Windows
````
e                       # Open the editor
//                      # Opens the date picker
shift + enter           # Insert a hard break (<br>)
ctrl + b                # Bold
ctrl + i                # Italics
ctrl + u                # Underline
ctrl + shift + s        # Strikethrough
ctrl + shift + m        # Monospace
ctrl + x                # Cut
ctrl + c                # Copy
ctrl + v                # Paste
ctrl + z                # Undo
ctrl + y                # Redo  
ctrl + shift + 7        # Numbered list
ctrl + shift + 8        # Bulleted list
ctrl + Left Alt + 0     # Set paragraph style
ctrl + Left Alt + 1-6   # Set as heading 1-6
ctrl + shift + 9        # Quote
ctrl + enter            # Publish
ctrl + shift + e        # Preview
ctrl + m                # Insert files & images
ctrl + k                # Insert link
ctrl + shift + d        # Insert markup
ctrl + shift + a        # Insert macro
ctrl + shift + I        # Insert a 3x3 table
ctrl + shift + minus    # Insert a horizontal rule
ctrl + h                # Find / Replace
Alt + shift + ↑         # Insert table row before
Alt + shift + ↓         # Insert table row after
Alt + shift + →         # Insert column after
Alt + shift + ←         # Insert column before
tab                     # Indent item when in a list
                        # Move cursor to next cell in the table
shift + tab             # Outdent item when in a list
                        # Move cursor to previous cell in the table
ctrl + \                # Clear all formatting
ctrl + enter            # Publish page
````

#### Markdown
````
SYNTAX                  PLACE MARKDOWN                          RESULT
*italics*               Around the text                         italics
_italics_               Around the text                         italics
**bold**                Around the text                         bold
__bold__                Around the text                         bold
~~strikethrough~~       Around the text                         strikethrough
`monospace`             Around the text                         monospace
#                       Around the text                         h1
##                      Around the text                         h2
###                     Around the text                         h3
####                    Around the text                         h4
#####                   Around the text                         h5
######                  Around the text                         H6
*                       At the beginning of each list item      Unordered list
-                       At the beginning of each list item      Unordered list
1.                      At the beginning of each list item      Ordered list
2)                      At the beginning of each list item      Ordered list
>                       At the start of the block quote         Block quote 
---                     On a new line                           Horizontal rule
***                     On a new line                           Horizontal rule
(awthanks)              Around the emoji shortcut text          Emoji (awthanks)
```                     At the start of the code block          Code block
````
