#### BASICS
##### Heading
````
# H1
## H2
### H3
#### H4
##### H5
###### H6
````
##### Bold
````
**text bold**
````
##### Italic
````
*text italic*
````
##### Blockquote
````
> blockquote
````
##### Ordered List
````
1. Item One
2. Item Two
3. Item Three
4. ...
````
##### Unordered List
````
- Item One
- Item Two
- Item Three
- ...
````
##### Code
````
`code`
````
##### Horizontal Rule
````
--- HR
````
##### Link
````
[title](https://www.example.com)
````
##### Image
````
![alt text](image.png)
````
