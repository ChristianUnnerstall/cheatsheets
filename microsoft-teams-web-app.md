### General
````
CTRL+.                      # Show keyboard shortcuts<br>
CTRL+E                      # Go to Search<br>
CTRL+/                      # Show commands<br>
CTRL+SHIFT+G                # Goto<br>
ALT+N                       # Start a new chat<br>
CTRL+,                      # Open Settings<br>
CTRL+F1                     # Open Help<br>
Esc                         # Close
````
### Navigation
````
CTRL+SHIFT+1                # Open Activity<br>
CTRL+SHIFT+2                # Open Chat<br>
CTRL+SHIFT+3                # Open Teams<br>
CTRL+SHIFT+4                # Open Meetings<br>
CTRL+SHIFT+5                # Open Calls<br>
CTRL+SHIFT+6                # Open Files<br>
ALT+Up arrow key            # Go to previous list item<br>
ALT+Down arrow key          # Go to next list item<br>
CTRL+SHIFT+F6               # Go to previous section<br>
CTRL+F6                     # Go to next section
````
### Messaging
````
C                           # Go to compose box<br>
CTRL+SHIFT+X                # Expand compose box<br>
CTRL+Enter                  # Send (expanded compose box)<br>
CTRL+SHIFT+O                # Attach file<br>
SHIFT+Enter                 # Start new line<br>
R                           # Reply to thread<br>
CTRL+SHIFT+I                # Mark as important
````
### Meetings and Calls
````
CTRL+SHIFT+A                # Accept video call<br>
CTRL+SHIFT+S                # Accept audio call<br>
CTRL+SHIFT+D                # Decline call<br>
CTRL+SHIFT+C                # Start audio call<br>
CTRL+SHIFT+U                # Start video call<br>
CTRL+SHIFT+M                # Toggle mute<br>
CTRL+SHIFT+F                # Toggle fullscreen<br>
CTRL+SHIFT+Space            # Go to sharing toolbar
````
