#### General
````
WINDOWS                         # open or close start menu
WONDOWS + A                     # open action center
WINDOWS + C                     # open cortana in listening mode
WINDOWS + D                     # display and hide the desktop
WINDOWS + E                     # open file explorer
WINDOWS + G                     # open game bar when a game is open
WINDOWS + H                     # open the share charm
WINDOWS + I                     # open settings
WINDOWS + K                     # open the connect quick action
WINDOWS + L                     # lock your PC or switch accounts
WINDOWS + M                     # minimize all windows
WINDOWS + R                     # open run dialog box
WINDOWS + S                     # open search
WINDOWS + U                     # open ease of access center
WINDOWS + X                     # open quick link menu.
WINDOWS + NUMBER (e.g. 1,2,..)  # open the app pinned to the taskbar in the position indicated by the number
WINDOWS + LEFT                  # snap app windows left
WINDOWS + RIGHT                 # snap app windows right
WINDOWS + UP                    # maximize app windows
WINDOWS + DOWN                  # minimize app windows
WINDOWS + ,                     # temporarily peek at the desktop
WINDOWS + CTRL + D              # aAdd a virtual desktop
WINDOWS + CTRL + LEFT/RIGHT     # switch between virtual desktops
WINDOWS + CTRL + F4             # cClose current virtual desktop
WINDOWS + ENTER                 # open Narrator
WINDOWS + HOME                  # minimize all but the active desktop window (restores all windows on second stroke)
WINDOWS + PRINT                 # capture a screenshot and save in screenshots folder
WINDOWS + SHIFT + UP            # stretch the desktop window to the top and bottom of the screen
WINDOWS + TAB                   # open task view
WINDOWS + +                     # zoom in using the magnifier
WINDOWS + -                     # zoom out using the magnifier
CTRL + SHIFT + ESC              # open task manager
Alt + TAB                       # switch between open apps
Alt + LEFT                      # go back
Alt + RIGHT                     # go foward
Alt + PAGE UP                   # move up one screen
Alt + PAGE DOWN                 # move down one screen
CTRL + ALT + TAB                # view open apps
CTRL + C                        # copy selected items to clipboard
CTRL + X                        # cut selected items
CTRL + V                        # paste content from clipboard
CTRL + A                        # select all content
CTRL + Z                        # undo an action
CTRL + Y                        # redo an action
CTRL + D                        # delete the selected item and move it to the recycle bin
CTRL + ESC                      # open the start menu
CTRL + SHIFT                    # switch the keyboard layout
CTRL + SHIFT + ESC              # open task manager
CTRL + F4                       # close the active window
````
