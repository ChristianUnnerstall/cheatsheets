### General
````
CTRL+.                      # Show keyboard shortcuts<br>
CTRL+E                      # Go to Search<br>
CTRL+/                      # Show commands<br>
CTRL+G                      # Goto<br>
CTRL+N                      # Start a new chat<br>
CTRL+,                      # Open Settings<br>
F1                          # Open Help<br>
Esc                         # Close<br>
CTRL+=                      # Zoom in<br>
CTRL+-                      # Zoom out
````
### Navigation
````
CTRL+1                      # Open Activity<br>
CTRL+2                      # Open Chat<br>
CTRL+3                      # Open Teams<br>
CTRL+4                      # Open Meetings<br>
CTRL+5                      # Open Calls<br>
CTRL+6                      # Open Files<br>
ALT+SHIFT+Up arrow key      # Go to previous list item<br>
ALT+SHIFT+Down arrow key    # Go to next list item<br>
CTRL+SHIFT+Up arrow key     # Move selected team up<br>
CTRL+SHIFT+Down arrow key   # Move selected team down<br>
CTRL+SHIFT+F6               # Go to previous section<br>
CTRL+F6                     # Go to next section
````
### Messaging
````
C                           # Go to compose box
CTRL+SHIFT+X                # Expand compose box
CTRL+Enter                  # Send (expanded compose box)
CTRL+O                      # Attach file
SHIFT+Enter                 # Start new line
R                           # Reply to thread
CTRL+SHIFT+I                # Mark as important
````
### Meetings and Calls
````
CTRL+SHIFT+A                # Accept video call<br>
CTRL+SHIFT+S                # Accept audio call<br>
CTRL+SHIFT+D                # Decline call<br>
CTRL+SHIFT+C                # Start audio call<br>
CTRL+SHIFT+U                # Start video call<br>
CTRL+SHIFT+M                # Toggle mute<br>
CTRL+SHIFT+O                # Toggle video<br>
CTRL+SHIFT+F                # Toggle fullscreen<br>
CTRL+SHIFT+Space            # Go to sharing toolbar
````
