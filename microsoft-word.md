#### General
````
ALT + Q                 # go to "Tell me what you want to do"
CTRL + O                # open
CTRL + S                # save
CTRL + W                # close
CTRL + X                # cut
CTRL + C                # copy
CTRL + V                # paste
CTRL + A                # select all
CTRL+  B                # bold
CTRL + I                # italic
CTRL + U                # underline
CTRL + [                # decrease font size 1 point
CTRL + ]                # increase font size 1 point
CTRL + E                # center text
CTRL + L                # left align text
CTRL + R                # right align text
ESC                     # cancel
CTRL + Z                # undo
CTRL + Y                # redo
````
